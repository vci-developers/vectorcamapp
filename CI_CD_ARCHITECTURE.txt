GitHub Actions CI/CD Workflow Architecture
==========================================

────────────────────────────

    ┌──────────────────────────────────────────────────────────────────┐
    │                    Trigger Events                                │
    ├──────────────────┬──────────────────┬───────────────────────────┤
    │   Push to main   │  Pull Request    │  Manual Trigger          │
    │   Push to dev    │  (any branch)    │  (workflow_dispatch)     │
    └────────┬─────────┴────────┬─────────┴────────────┬──────────────┘
             │                  │                       │
             ├──────────────────┼───────────────────────┤
             │                  │                       │
    ┌────────▼─────────┐  ┌────▼──────────┐  ┌────────▼─────────┐

    └──────────────────┘  └───────────────┘  └──────────────────┘


🔄 Build Matrix Execution (Parallel)

    Start Build

    └────┬────┘        └────┬────┘       └────┬────┘       Execution
         │                  │                  │               │
    ┌────▼────┐        ┌────▼────┐       ┌────▼────┐           │
    │  Build  │        │  Build  │       │  Build  │           │
    │   APK   │        │   APK   │       │   APK   │           │

         │                  │                  │                │
    ┌────▼────┐        ┌────▼────┐       ┌────▼────┐           │
    │  Run    │        │  Run    │       │  Run    │           │
    │  Tests  │        │  Tests  │       │  Tests  │           │
    └────┬────┘        └────┬────┘       └────┬────┘           │
         │                  │                  │                │
    ┌────▼────┐        ┌────▼────┐       ┌────▼────┐           │
    │ Upload  │        │ Upload  │       │ Upload  │           │

🎯 Recommended Usage
────────────────────

    Development Branch (develop):
    ├─ PR opened
    │  └─► pr-check.yml runs (fast validation)
    │
    ├─ PR merged to develop
    │  └─► build-debug.yml runs (simple build)
    │
    └─ Push to develop
       └─► build-debug.yml runs

    Main Branch (main):
    ├─ PR opened
    │  └─► pr-check.yml runs
    │
    ├─ PR merged to main
    │  └─► build-and-distribute.yml runs
    │      ├─ Builds APKs
    │      ├─ Runs tests
    │      └─► Distributes to Firebase 🚀
    │
    └─ Manual deployment needed
       └─► Trigger build-and-distribute.yml manually
